# Sentinel Pro

**Sentinel Core is free and open** ([Apache-2.0](LICENSE)) — clone it, build it, train it,
run it forever at no cost. Everything needed to use Sentinel is in this repo.

**Sentinel Pro** is for teams and individuals who want the result without the work of
building, training, and maintaining it themselves. It is a separate, commercial offering.

## What Pro adds

- **🥧 Ready-to-flash Raspberry Pi image.** Burn it to an SD card and boot — no compiling,
  no training, no corpus fetching. Sentinel is running in minutes.
- **🧠 Pretrained large model.** The 101M-parameter checkpoint, trained on a large
  security/coding/CVE corpus, ships ready to use (the free core trains its own).
- **📡 Auto-updating threat intelligence.** A daily-refreshed CVE / malware / OSINT feed
  so retrieval always reflects the latest vulnerabilities — fully offline after sync.
- **📊 Dashboard + reporting.** A polished web console with scheduled scans, alerts, and
  PDF/JSON export for handing findings to a client or a ticketing system.
- **🛟 Priority support & deployment help.** Direct help standing it up in your
  environment, plus an SLA for response times.

## Who it's for

Consultancies and MSSPs deploying to many clients · IR teams that need it running *now* ·
home-lab and self-hosting users who'd rather flash an image than compile C · educators
running it across a classroom.

## How it works (open-core)

The **core stays open** — you can always build and run it yourself for free. Pro is the
**convenience + maintenance + advanced features** layer on top, delivered as a prebuilt
image and a maintained data/feature service. You're paying for the done-for-you packaging
and ongoing updates, not for permission to use the core.
